POS Fixed Amount Discount and Percentage Discount Odoo Apps

This Odoo apps allow your customer to fixed amount discount and Percentage Discount option with configurable option Odoo point of sale. POS user can easily apply configurable discount option on POS Order from pos screen and applied discount print on pos receipt.

Features

Configure Multiple Discount Type.

Provide multiple discount type, also user can choose discount type in back-end then apply in pos order.

Discount Button on POS Screen.

User can Apply Discount by Discount Button on POS Screen.

Fixed Amount Discount.

User can Configure Fixed Amount Discount in POS Back-end.

Discount in POS Receipt & Backend POS Order.

User can print pos payment receipt with discount.

Percentage Amount Discount.

User can configure Percentage Amount Discount in POS Back-end.

POS Order with Fixed Discount.

User can see pos order details with fixed discount in back-end.

Dynamically Add Discount from POS Screen.

User can enter discount amount dynamically from POS Screen.

POS Order with Percentage Discount.

User can see pos order details with percentage discount back-end.
