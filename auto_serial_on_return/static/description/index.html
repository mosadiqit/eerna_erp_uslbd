User Guide

Create an SO and validate the DO (notice the lot No. in the 'Detailed Operations' wizard).

Once the DO is validated, click on 'Return' to return any products that might have been faulty or not up to the customer's expectations,...etc.

Notice that when you open the 'Detailed Operations' wizard for lots registration, the lot numbers for returend products is automatically specified for you to just confirm and then validate the record accordingly (if the products were originally taken from more than one lot, then the lots on the 'Detailed Operations' wizard will be filtered on return to choose only from them accordingly).
